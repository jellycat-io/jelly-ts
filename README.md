# Jelly Engine

A 2D game engine using WebGL2 written in Typescript

## How to use

Install dependencies
```
yarn
```

Launch sample game
```
yarn dev
```

Build documentation
```
yarn docs:build
```

Launch documentation in browser
```
yarn docs:watch
```
